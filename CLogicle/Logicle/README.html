Classes Logicle and FastLogicle

Provide an implementation in C++ of the Logicle data scale and its inverse. The Logicle data scale is a generalization of the inverse hyperbolic sine function or arcsinh. Like arcsinh it is logarithmic for large data values, linear in the neighborhood of zero, symmetric around zero and changes smoothly between the log and linear regions. It is more general because the width of the linear region can be independently controlled. The scale is parameterized by the choice of a number of decades of data range to cover, the maximum data value on scale and the width of the linear region. It is used in flow cytometry where a large dynamic range of data, most suitable for a logarithmic scale, is present along with other data distributed near zero and including negative values, for which an ordinary logarithmic scale is not suitable.

Pure C Logicle

An implementation of the Logicle scale using only ANSI standard C. The functionality is the same as the C++ version but due to the limitations of the C language it is more awkward to use. However, it can be called more easily from other languages, for example, R or MATLAB and does not require linking with any C++ libraries.

Test Logicle

A set of unit test for the Logicle implementation using the Google Test framework, an implementation of the xUnit test methods.
