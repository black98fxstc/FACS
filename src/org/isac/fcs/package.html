Utilities for reading and writing Flow Cytometry Standard (FCS) data files as defined by the International Society for Analytical Cytology (ISAC)

Features of FCS that are not supported

  • ASCII data $DATATYPE=A
  • Double precision floating point data $DATATYPE=D
  • Uncorrelated univariate histograms $MODE=U
  • Correlated multivariate histograms $MODE=C
  • Cyclic Redundancy Check (CRC)
  • OTHER segments
  • Multiple datasets per File, that is $NEXTDATA is always 0
